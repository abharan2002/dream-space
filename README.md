# DreamSpace | Personal Garage

A private automotive archive shaped by curation, preservation, and precision.

## Overview

DreamSpace is a web application built to showcase a curated automotive collection. It features a refined user interface with custom interactions, smooth scrolling, and seamless page transitions, delivering a premium browsing experience.

## Features

- **Custom Interactions**: Includes a tailored custom cursor for enhanced user experience.
- **Smooth Scrolling**: Implements smooth scrolling mechanics for a fluid browsing feel.
- **Page Transitions**: Seamless transitions between views.
- **Modern Tech Stack**: Built with Next.js (App Router), React, TypeScript, and Tailwind CSS.
- **Custom Typography**: Utilizes Cormorant Garamond for display and Manrope for body text.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/) / Custom Animation Libraries (implied by transitions and smooth scroll)

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components (CustomCursor, SmoothScroll, PageTransition, etc.).
- `public`: Static assets including images and icons.
